7 – Modelado 3D de superficies

Para este encargo se nos pidió escoger 2 tutoriales complementarios, el segundo tutorial que escogí es el de Modelado 3D de superficies. En donde trabajaremos con el entorno de modelado de Surface del programa Fusion, con el objetivo de crear un sólido a partir de una serie de superficies.

Pasos a seguir para la creación de un sólido

Fecha: 17-08-2021

Paso 1:

Lo primero que vamos a realizar es un Sketch en el plano Top en la sección de Surface, y utilizaremos la Spliney realizaremos una geometría simple.

Paso 2:

Lo siguiente es hacer aplicar el comando Plano de desface sobre la cara del Sketch que hicimos en el paso anterior, desfasándolo unos 110 mm. hacia arriba. Y sobre ese nuevo plano de construcción vamos a realizar una figura parecida a la que tenemos en el plano inferior.

Paso 3:

Ocuparemos el comando Loftque se encuentra en el menú de superficies, para así unir las dos líneas que hemos creado anteriormente, y crear una nueva superficie.

Paso 4:

Vamos a realizar un nuevo Sketch, pero esta vez en la vista frontal. Realizaremos otra figura simple.

Paso 5:

A la figura que creamos recién le aplicaremos un Extrudehacía ambos lados, dejando pasada la geometría de la superficie anterior.

Paso 6:

Lo que queremos lograr es crear un agujero en la primera superficie en donde se intersecta con la segunda superficie. Para esto vamos a aplicar un Split Body.

Paso 7:

Queremos que de la segunda superficie quede solamente lo que se encuentra al interior de la primero superficie. Para esto debemos aplicar un Trim.

Paso 8:

Vamos a utilizar el comando Patchpara cerrar las caras de nuestra superficie.

Paso 9:

Paso siguientes es unir las superficies para lograr obtener un sólido, para esto utilizaremos en comando ``Stitch```.

Nota: Que las líneas estén de color verde significa que la toda la cara se unira bien con la otra superficie.

Paso 10:

Buscamos redondear un poco los bordes de la figura, para esto usaremos Fillet.

Paso 11:

Utilizaremos el comando Shell para generar un vaciado, es para que crear como una cajaj en donde los bordes tienen un determinado grosor, en este caso un grosor de 3 mm.
