Post 5 – Impresión 3D

Para este encargo se nos pidió seguir el video tutorial de Impresión 3D con el programa Cura.

Qué es Cura?

Fecha: 17-08-2021

Las impresiones 3D necesitan de un software de corte, también conocido como laminador, para convertir el modelo 3D en un archivo de código G que dará a la impresora 3D todas las instrucciones necesarias. Cura es uno de estos programas, es un software de código abierto. Una de las ventajas de Cura es su facilidad de uso, soporte para diferentes formatos de archivos y compatibilidad con muchas impresoras 3D.

Cómo utilizar el programa?

Paso 1

Lo primero que debemos realizar es configurar el tipo de impresora, el material y el cabezal que vamos a utilizar. En este caso vamos seleccionar la impresora 3D Creality Ender – 3, esta trabaja de manera off-line.

Paso 2

Lo siguiente es ir al material y cabezal, para este ejercicio trabajaremos con PLA+ como material y el cabezal será de 0,4 mm.

Paso 3

Luego exportamos el archivo que contiene el modelo que queremos imprimir, en este caso yo utilicé un jarrón que hice en el curso de Taller integrado del primer trimestre. Hay que asegurarse que el modelo esté en una buena posición, puesto que al abrir el archivo en el Cura puede que aparezca en una orientación diferente.

Además, hay que modificar algunos ítems de configuración:

  1. Agregarle los soportes, estos son los de color celeste.
  2. La densidad del relleno sea de 6.0%
  3. La altura de la capa este en 0.3 mm.
